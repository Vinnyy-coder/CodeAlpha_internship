#  Unemployment Analysis with Python

##  Project Overview

This project was completed as part of the **CodeAlpha Data Science Internship**.

The objective of this project is to analyze unemployment trends in India using Python and data visualization techniques. The analysis focuses on understanding unemployment patterns across different regions, studying the impact of COVID-19, and identifying trends that can help in economic and social decision-making.

---

##  Objectives

* Analyze unemployment rate data in India.
* Perform data cleaning and preprocessing.
* Explore unemployment trends using visualization techniques.
* Study the impact of COVID-19 on unemployment rates.
* Compare unemployment patterns across regions and areas.
* Generate meaningful insights from the dataset.

---

##  Dataset Information

Dataset Used: **Unemployment in India.csv**

Dataset Features:

* Region
* Date
* Frequency
* Estimated Unemployment Rate (%)
* Estimated Employed
* Estimated Labour Participation Rate (%)
* Area (Urban/Rural)

## 📈 Key Findings

* Significant variation in unemployment rates across different states.
* Urban and rural areas displayed different unemployment patterns.
* COVID-19 had a noticeable impact on unemployment rates.
* Certain regions consistently showed higher unemployment rates.
* Labour participation and employment metrics influenced unemployment trends.
