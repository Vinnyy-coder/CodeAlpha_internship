#  Iris Flower Classification using Machine Learning

## 📌 Project Overview

This project focuses on classifying Iris flower species using Machine Learning. The Iris dataset is one of the most popular datasets for classification tasks and contains measurements of iris flowers belonging to three different species:

* Iris Setosa
* Iris Versicolor
* Iris Virginica

A Random Forest Classifier was trained on the dataset to accurately predict flower species based on their physical measurements.

## 📂 Dataset Information

The dataset contains 150 flower samples with the following features:

| Feature      | Description           |
| ------------ | --------------------- |
| Sepal Length | Length of sepal in cm |
| Sepal Width  | Width of sepal in cm  |
| Petal Length | Length of petal in cm |
| Petal Width  | Width of petal in cm  |

### Target Classes

* Setosa
* Versicolor
* Virginica
